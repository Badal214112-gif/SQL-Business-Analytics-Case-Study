# SQL Business Analytics Case Study – Staff Data Analysis

---

## Project Overview

This project focuses on analyzing employee (staff) data using SQL to generate meaningful business insights. The analysis simulates real-world Business Analyst tasks where structured data is explored to identify patterns, inefficiencies, and opportunities for improvement.

The project covers end-to-end SQL analysis, from basic data exploration to advanced querying techniques.

---

## Dataset Description

The dataset contains employee-level information, including:

* Employee ID
* First Name & Last Name
* Department
* Salary
* Job Role
* Commission

This type of dataset is commonly used in HR analytics to evaluate workforce distribution and compensation structure.

---

## Objectives

* Analyze employee distribution across departments
* Evaluate salary trends and variations
* Identify top-performing and high-earning employees
* Detect salary imbalances and outliers
* Apply advanced SQL techniques to solve business problems

---

## Approach

### 1. Data Understanding

* Reviewed dataset structure and column definitions
* Identified key attributes such as salary, department, and roles

---

### 2. Data Cleaning

* Checked for NULL values
* Removed duplicate records
* Validated salary and commission values

---

### 3. SQL Analysis

#### Basic Analysis

* Total number of employees
* Average salary across the organization
* Employee count by department

#### Intermediate Analysis

* Highest and lowest salary by department
* Average salary by job role
* Employees earning above average salary

#### Advanced Analysis

* Department-wise salary ranking using window functions
* Identification of top earners in each department
* Salary distribution analysis
* Use of subqueries for comparative insights

---

## SQL Concepts Used

* SELECT, WHERE, ORDER BY
* GROUP BY & Aggregations (COUNT, AVG, SUM)
* Joins
* Subqueries
* Window Functions (RANK, ROW_NUMBER, PARTITION BY)
* CASE WHEN (conditional logic)

---

## Key Insights

* Salary distribution varies significantly across departments
* A small group of employees contributes to higher salary concentration
* Some employees earn significantly above department averages
* Certain roles may be underpaid compared to others

---

## Business Recommendations

* Standardize salary structure across departments
* Introduce performance-based incentives
* Review compensation for outliers
* Optimize workforce distribution across roles

---

## Future Scope

* Implement A/B testing for compensation strategies
* Build dashboards using BI tools (Power BI / Tableau)
* Integrate additional employee performance data for deeper analysis

This project demonstrates how SQL can be effectively used to analyze structured data and generate actionable business insights. It reflects core Business Analyst responsibilities such as data exploration, problem-solving, and decision support.
